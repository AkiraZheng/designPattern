## 3. 附加说明

由于本项目中实现的设计模式代码较为简单，因此为了方便阅读，我将所有类的的实现都放在了头文件中，这样可以方便的查看每个类的实现。

同时为了方便测试，我在每个设计模式的实现中都写了一个测试函数，可以直接运行测试函数来测试每个设计模式的实现。测试函数的实现在`main.cpp`中。
